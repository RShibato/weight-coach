# WEIGHT COACH

**科学的な減量管理とメンタルサポート**

格闘技選手のための体重管理アプリケーション

## 🥊 主な機能

### 📊 精密な体重管理
- 水分補正機能で正確な体重推移を把握
- 体重・体脂肪率の2軸グラフ表示
- 試合日までのカウントダウン

### 🍽️ 栄養管理
- 100品目以上の食品データベース
- PFCバランスを円グラフで可視化
- 目標体重に基づいた推奨PFC値の表示

### 💧 水分管理
- 1日の水分摂取量を記録
- 過去7日間のグラフ表示

### 🧠 メンタルサポート
- 毎日のコンディション記録
- コーチからの励ましメッセージ

### 📈 その他の機能
- 適正階級診断
- オンボーディング機能
- データのリセット
