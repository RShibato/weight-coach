# WEIGHT COACH

**科学的な減量管理とメンタルサポート**

格闘技選手のための体重管理アプリケーション

## 🥊 主な機能

### 📊 精密な体重管理
- 水分補正機能で正確な体重推移を把握
- 体重・体脂肪率の2軸グラフ表示
- 試合日までのカウントダウン

### 🍽️ 栄養管理
- 100品目以上の食品データベース
- PFCバランスを円グラフで可視化
- 目標体重に基づいた推奨PFC値の表示
- 朝昼夜の食事タイミング記録

### 💧 水分管理
- 1日の水分摂取量を記録
- 過去7日間のグラフ表示
- +250ml / +500ml / -250ml の便利ボタン

### 🧠 メンタルサポート
- 毎日のコンディション記録
- コーチからの励ましメッセージ
- 5段階の気分選択

### 📈 その他の機能
- 適正階級診断（身長・体脂肪率・骨格から分析）
- オンボーディング機能（初回利用時のガイド）
- メールマガジン登録
- データのエクスポート/リセット

## 📦 インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## 🎯 使い方

1. 新規登録またはログイン
2. オンボーディングで基本情報を入力
3. 毎日の体重・食事・水分を記録
4. グラフで推移を確認
5. 目標達成！

## 📱 対応デバイス

- デスクトップ（Chrome, Firefox, Safari, Edge）
- モバイル（iOS Safari, Chrome）
- タブレット

## 🔒 データ保存

すべてのデータはブラウザのLocalStorageに保存されます。
サーバーには送信されないため、プライバシーが保護されます。
